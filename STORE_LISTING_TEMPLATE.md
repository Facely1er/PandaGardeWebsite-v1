## Tips for Writing Store Listing

### Do's ✅
- Use clear, concise language
- Highlight key features with bullet points
- Include benefits, not just features
- Use emojis sparingly (they can help readability)
- Proofread for typos
- Keep descriptions accurate
- Update regularly with new features

### Don'ts ❌
- Don't use excessive capitalization
- Don't make false claims
- Don't use misleading screenshots
- Don't copy from other apps
- Don't use spam keywords
- Don't forget to update release notes
