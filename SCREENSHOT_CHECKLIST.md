## ✅ Quality Checklist

For each screenshot, verify:

- [ ] Screenshot is clear and readable
- [ ] Text is visible and not cut off
- [ ] Shows key features of the app
- [ ] No sensitive/personal data visible (or blurred)
- [ ] Saved as PNG format
- [ ] Saved to: `play-store-assets/screenshots/`
- [ ] Named correctly (screenshot-1.png, etc.)

---

## 📐 Size Requirements

**Recommended**: 1080x1920 pixels (9:16 portrait)

**If different size**:
- Resize using: https://www.iloveimg.com/resize-image
- Or image editor

---

## 🎯 Quick Steps in Android Studio

1. ✅ Emulator running
2. ✅ App launched
3. ✅ Navigate to screen
4. ✅ Capture screenshot (camera icon)
5. ✅ Save to `play-store-assets/screenshots/`
6. ✅ Repeat for next screen
