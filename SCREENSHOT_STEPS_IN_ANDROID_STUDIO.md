# 📸 Screenshot Steps in Android Studio - Follow These

**Android Studio is open - follow these exact steps:**

---

## 🎯 Step-by-Step Instructions

### Step 1: Start Emulator (If Not Running)

1. **In Android Studio**, look at the top toolbar
2. Click **Device Manager** icon (or View → Tool Windows → Device Manager)
3. If you see a device listed:
   - Click the **Play button** (▶) next to it to start
4. If no device exists:
   - Click **Create Device**
   - Choose: **Phone → Pixel 5** (or Pixel 6)
   - Click **Next**
   - Choose system image (latest Android, API 33+)
   - Click **Next → Finish**
   - Click **Play** to start emulator
5. **Wait** for emulator to boot (may take 1-2 minutes)

---

### Step 2: Run Your App

1. **In Android Studio**, look at the top toolbar
2. Find the **green Play button** (▶) - usually top right
3. **Select your app** from the dropdown (should show "app")
4. **Click the green Play button**
5. **Wait** for app to build and install (may take 1-2 minutes)
6. App should launch in the emulator

---

### Step 3: Navigate to Screens

**Once app is running in emulator:**

1. **Family Dashboard** (main screen)
   - This should be the first screen
   - If not, navigate to it

2. **Child Progress** screen
   - Click on a family member or progress section
   - Navigate to detailed progress view

3. **Learning Hub** screen
   - Navigate to games/activities section

4. **Privacy Goals** screen
   - Navigate to goals management

5. **Family Members** screen
   - Navigate to family management

---

### Step 4: Capture Screenshots

**For each screen:**

1. **Navigate to the screen** you want to capture
2. **In the emulator toolbar** (top of emulator window):
   - Look for a **camera icon** 📷
   - Or: **three dots menu** (⋮) → **Screenshot**
3. **Click the camera icon**
4. Screenshot is captured automatically
5. **Save dialog appears** - save to:
   - `C:\Users\facel\Downloads\GitHub\PandaGardeWebsite-v1\play-store-assets\screenshots\`
   - Name as: `play-store-screenshot-1.png`, `play-store-screenshot-2.png`, etc.

**Repeat for each screen** (minimum 2, recommended 4-5)

---

### Step 5: Alternative - Device File Explorer

**If camera icon doesn't work:**

1. In Android Studio: **View → Tool Windows → Device File Explorer**
2. Navigate to: `/sdcard/Pictures/Screenshots/`
3. You'll see screenshot files
4. **Right-click** on each screenshot
5. **Save As** → Save to `play-store-assets/screenshots/`

## 🆘 Troubleshooting

**"No emulator running"**
- Start emulator from Device Manager
- Wait for it to fully boot

**"App won't run"**
- Check for build errors in Android Studio
- Try: Build → Clean Project, then Build → Rebuild Project
- Then run again

**"Can't find camera icon"**
- Use: View → Tool Windows → Device File Explorer
- Or: Three dots menu (⋮) in emulator → Screenshot

**"Screenshots are wrong size"**
- That's okay - you can resize later
- Use: https://www.iloveimg.com/resize-image
- Resize to 1080x1920
