# 📸 Play Store Assets Folder

**Store your Play Store submission assets here**

---

## 📁 Folder Structure

```
play-store-assets/
├── screenshots/          # App screenshots
├── icon-512.png          # App icon (512x512)
└── feature-graphic.png   # Feature graphic (1024x500)
```

---

## 📸 Screenshots

**Save screenshots here:**
- `screenshots/play-store-screenshot-1.png`
- `screenshots/play-store-screenshot-2.png`
- `screenshots/play-store-screenshot-3.png`
- etc.

**Requirements**:
- Size: 1080x1920 (9:16 portrait) recommended
- Format: PNG
- Minimum: 2 screenshots
- Maximum: 8 screenshots

---

## 🎨 App Icon

**Save as**: `icon-512.png` (or `play-store-icon-512.png`)

**Requirements**:
- Size: 512x512 pixels
- Format: PNG
- Background: Transparent or solid

---

## 🖼️ Feature Graphic

**Save as**: `feature-graphic.png` (or `play-store-feature-graphic-1024x500.png`)

**Requirements**:
- Size: 1024x500 pixels
- Format: PNG
- Content: App name, tagline, features
