# 🎯 Do This in Android Studio - Step by Step

**Android Studio is open - follow these exact steps:**

---

## ✅ Step 1: Start Emulator

**In Android Studio:**

1. Look at the **left sidebar** - find **Device Manager** tab
   - If not visible: **View → Tool Windows → Device Manager**

2. **If you see a device listed:**
   - Click the **▶ Play button** next to it
   - Wait for emulator to start (1-2 minutes)

3. **If no device exists:**
   - Click **Create Device** button
   - Select: **Phone → Pixel 5** (recommended)
   - Click **Next**
   - Select system image (choose latest, API 33+)
   - Click **Next → Finish**
   - Click **▶ Play** to start

---

## ✅ Step 2: Run Your App

**In Android Studio:**

1. Look at the **top toolbar** - find the **green Play button** (▶)
2. Make sure **"app"** is selected in the dropdown next to it
3. **Click the green Play button**
4. Wait for build to complete (may take 1-2 minutes)
5. App will launch in the emulator automatically

---

## ✅ Step 3: Capture Screenshots

**Once app is running in emulator:**

### For Each Screen:

1. **Navigate to the screen** in the app:
   - Family Dashboard (main screen)
   - Child Progress (click on a member)
   - Learning Hub (games section)
   - Privacy Goals (goals screen)
   - Family Members (management)

2. **Capture screenshot:**
   - Look at the **emulator window toolbar** (top of emulator)
   - Find the **📷 camera icon** (or three dots menu)
   - **Click the camera icon**
   - Screenshot is captured

3. **Save screenshot:**
   - Save dialog will appear
   - Navigate to: `C:\Users\facel\Downloads\GitHub\PandaGardeWebsite-v1\play-store-assets\screenshots\`
   - Name as: `play-store-screenshot-1.png` (then 2, 3, etc.)
   - Click **Save**

4. **Repeat** for each screen (minimum 2, recommended 4-5)

**Alternative method:**

1. In Android Studio: **View → Tool Windows → Device File Explorer**
2. Navigate to: `/sdcard/Pictures/Screenshots/`
3. You'll see your screenshots
4. **Right-click** each one → **Save As**
5. Save to: `play-store-assets/screenshots/`
