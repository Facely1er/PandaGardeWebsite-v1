## 🎯 Quick Start

### Build and Test (5 minutes)

```bash
# 1. Build web app
npm run build

# 2. Sync to Android
npx cap sync android

# 3. Open in Android Studio
npx cap open android
```

Then in Android Studio: **Build → Build Bundle(s) / APK(s) → Build APK(s)**

### Or Use Build Script

```bash
# Windows
.\scripts\build-android.ps1

# Linux/macOS
./scripts/build-android.sh
```
